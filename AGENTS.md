# AGENTS.md

## Repository Overview

DCGAN implementation in TensorFlow 2 with training GIF generation. Single-purpose ML project with four Python modules: dataset loading, generator, discriminator, and training loop.

## Code Guidelines

- Python 3.10+
- Follow existing code style (TensorFlow 2 / Keras patterns)
- Lint with `ruff check .` before committing
- No secrets or API keys in code

## PR Guidelines

- All changes via pull request
- Squash merge only
